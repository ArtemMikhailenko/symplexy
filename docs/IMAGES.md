# Інструкції по зображеннях для Hero секції

## Необхідні зображення

Розмістіть наступні файли в папці `/public/images/`:

### 1. hero.png
**Опис:** Головне зображення дівчини з AI елементами  
**Розмір:** ~1110x833px  
**Формат:** PNG з прозорістю  
**Позиція:** Праворуч від тексту  

### 2. hero-cloude.png
**Опис:** Декоративні хмари для фону  
**Розмір:** ~976x688px та ~926x605px  
**Формат:** PNG з прозорістю  
**Використання:** 2 екземпляри (ліворуч і праворуч)  
**Opacity:** 60% для м'якшого ефекту

### 3. hero-bubbles.png
**Опис:** Декоративні бульбашки/сфери для фону  
**Формат:** PNG з прозорістю  
**Opacity:** 40-60% залежно від екрану

## Логотипи брендів

Замініть placeholder SVG файли в `/public/images/brands/` на реальні логотипи:

- socialbee.svg
- frase.svg
- leonardo.svg
- durable.svg
- heygen.svg
- descript.svg

**Рекомендації:**
- Формат: SVG або PNG (з прозорим фоном)
- Розмір: ~120-160px ширина
- Колір: Чорно-білі або оригінальні (grayscale ефект додається автоматично)

## Оптимізація

Next.js автоматично оптимізує зображення через компонент `next/image`:
- WebP конвертація
- Lazy loading
- Responsive sizes
- Priority для critical images
